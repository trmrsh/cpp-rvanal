rvanal software change log

This file to records changes to the 'rvanal' software. Each entry has a date, a list of affected components and comments. Look here to see what might have changed since your version of the software.

02/01/08 aliases.cc
Version 1.0.7 		Updates for withdrawal of SOFA
21/08/06 Version 1.0.5 Update for return of the server
10/04/06 dprob.cc Added output of parameters.
22/03/06 Lots
Version 1.0.4 		Portability changes. rvanal.h to trm_rvanal.h, and corresponding subs-related ones.
26/08/05 Installation
Version 1.0.3 		Major changes to the installation could well not work.
11/08/05 pgram.cc Added a few more arguments to allow more control over what is plotted. Fixed a minor bug.
10/08/05 aliases.cc Put in option to scale to effectively make reduced chi**2 = 1 in order that competing aliases are not artificially removed from consideration.
10/08/05 lots Removed HJD specific stuff as far as possible. Now no particular time is implied as it does not matter for most routines. Exception is 'aliases' for which HMJD, i.e. heliocentric MJDs are expected. Also removed hidden nature of the oversampling parameter.
17/09/04 dprob.cc Changed from Array1D to Buffer1D. Fixed bug in subs function 'tchi' that was putting it an infinite loop for large numbers of points.
